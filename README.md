# Indonesia VDR (Virtual Data Room)

MIGAS - Oil & Gas Data Management Platform

This is a comprehensive Virtual Data Room system for managing Indonesia's oil and gas exploration and production data, featuring interactive maps, basin analytics, and infrastructure visualization.

## Features
- Interactive GIS mapping with ArcGIS SDK
- Oil & Gas platform and pipeline infrastructure visualization
- Basin analytics and performance metrics
- Document management and data cataloging
- User management and access control

## Commands
- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm start` - Start production server

## Technologies
- Next.js 15
- React 19
- ArcGIS Maps SDK for JavaScript
- TypeScript
- Tailwind CSS
